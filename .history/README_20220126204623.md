## General info
This project is simple Project ASP.NET Core MVC, it help user to arrange data formating coming from csv file.
## Technologies
Project is created with:
* React js
* React Hooks
* Css3
* jsx
* Backend is hosted by [chatengine.io](https://chatengine.io )

	
## Setup
To run this project, install it locally using npm:

```
$ cd ../chat-app
$ npm install
$ npm install --save-dev @babel/core @babel/eslint-parser
$ npm start
```
